Key Features

⚡ Fast Scanning

Quick heuristic analysis using keyword matching and regex patterns for rapid threat detection.

🔍 Deep Analysis

Uses pikepdf for comprehensive PDF parsing - extracts metadata, embedded files, and JavaScript actions.

🎯 IOC Extraction

Automatically extracts URLs, IP addresses, and suspicious keywords from PDF content.

📊 Risk Scoring

Calculates threat score (0-100) with verdict: clean, suspicious, or malicious.

🔐 Security Focus

Detects JavaScript actions, OpenAction, EmbeddedFiles, and other malicious PDF features.

🌐 Multi-Interface

REST API, Web UI, and TUI support for flexible integration.

System Architecture

📁 services/analyzer

Core PDF analysis engine with fast & deep scanning capabilities.

🌐 apps/web

Browser-based web interface for PDF uploads and results display.

⌨️ services/analyzer/tui.py

Terminal UI for command-line PDF analysis.

Analysis Pipeline

  1. File Reception - Upload PDF via API or UI
  2. Hash Computation - Calculate SHA256 for identification
  3. Fast Scan - Keyword matching, IOC regex extraction
  4. Deep Scan (optional) - pikepdf parsing for metadata/JS
  5. Risk Scoring - Calculate verdict based on findings
  6. Response - Return JSON result to client

Security & Privacy

• No persistent storage of uploaded files

• Files processed in memory, discarded after analysis

• SHA256 hash used for identification without full file retention

• Designed for safe, transient analysis workflow
