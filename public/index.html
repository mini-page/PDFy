Local Development

# Install dependencies
pip install -e ./services/analyzer

# Start the API server
cd services/analyzer
uvicorn app.main:app --host 0.0.0.0 --port 8000

Test the API

curl -X POST http://localhost:8000/analyze/fast \
  -F "file=@your-pdf.pdf"

Docker Deployment

# Using docker-compose
docker-compose up -d

# Or build manually
docker build -t pdfy .
docker run -p 8000:8000 pdfy

🚀 Key Features

Fast Scan

Quick Threat Detection

Rapid heuristic analysis using keyword matching and regex patterns for instant threat identification.

Deep Scan

Comprehensive Analysis

Uses pikepdf for thorough PDF parsing - extracts metadata, embedded files, and JavaScript actions.

IOC Extraction

Automatically extracts URLs, IP addresses, and suspicious keywords from PDF content.

Risk Scoring

Calculates threat score (0-100) with clear verdicts: clean, suspicious, or malicious.

Multi-Interface

REST API for integration, Web UI for browser access, TUI for terminal usage.

Privacy Focused

No persistent storage. Files processed in memory, discarded after analysis.

🔌 API Reference

POST /api/analyze/fast
Quick scan with keywords and IOC extraction
POST /api/analyze/deep
Deep scan with metadata, embedded files, and JavaScript detection
GET /api/health
Health check endpoint

Example Request

curl -X POST http://localhost:8000/api/analyze/fast \
  -F "file=@sample.pdf"

Response Format

{
  "file_name": "document.pdf",
  "sha256": "abc123...",
  "keyword_hits": ["/JavaScript"],
  "iocs": { "urls": [...], "ips": [...] },
  "summary": {
    "verdict": "suspicious",
    "score": 65,
    "confidence": "medium"
  }
}
